# Clon de Google

Este proyecto es una practica de maquetacion web que recrea la pagina principal de Google usando solo HTML y CSS.

## Vista general

El objetivo de este proyecto es poner en practica conceptos basicos de:

- Estructura semantica con HTML5
- Estilizado con CSS
- Alineacion de elementos con Flexbox
- Maquetacion de una interfaz inspirada en un producto real

La interfaz incluye:

- Barra de navegacion superior
- Logo central
- Barra de busqueda
- Botones de accion principales
- Pie de pagina con enlaces informativos

## Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts para iconos Material Symbols
- Flexbox para la distribucion de elementos

## Estructura del proyecto

## Como usarlo

1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador.

Tambien puedes usar una extension como Live Server en Visual Studio Code para verlo en desarrollo.

## Proposito academico

Este proyecto fue desarrollado como ejercicio practico para reforzar habilidades de desarrollo web front-end, replicando una interfaz conocida para entender mejor su distribucion visual y estilos base.
